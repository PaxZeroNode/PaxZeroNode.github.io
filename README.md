# Pax Mutuara Public Site

Public support and services landing page for Pax Mutuara.

## Purpose

- create one coherent public entry point
- route supporters to Crowdfunder first
- route direct-support traffic to Venmo second
- route service buyers to the paid inquiry path
- distinguish clearly between supporting the project and hiring the work

- Crowdfunder is live as the main public support path
- Venmo `@paxmutuara` is the fallback direct-support path
- service offers are inquiry-driven
- no collected revenue is claimed here
